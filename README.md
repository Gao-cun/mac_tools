# mac_tools

`mac_tools` 用于集中开发我个人日常使用的 macOS 小应用。  
目标是：按需开发、轻量实用、可独立运行。

## 子项目总揽

| 子项目 | 说明 | 当前状态 | 入口 |
| --- | --- | --- | --- |
| `cut_copy` | 系统截图自动复制到剪贴板，并统一保存到默认目录，支持一键清除 | 已实现 | `/Users/zhaowenhao/code/mac_tools/cut_copy` |

## 目录约定

- 每个小应用作为一个独立子目录，包含自己的源码、脚本和文档。
- 每个子项目至少包含：
- `README.md`：功能、运行方式、验收说明。
- 可执行入口（如脚本或应用打包脚本）。
- 需要时包含测试目录，覆盖核心行为。

## 后续扩展

新增子项目后，在本文件的“子项目总揽”追加一行，保持总览可快速检索。
